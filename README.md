# S-FedProx: Secure Adaptive FedProx for Ghanaian Remote Patient Monitoring

Code repository for the research proposal "An Engineered Secure-FedProx Framework
for Privacy-Preserving Remote Patient Health Monitoring in Ghanaian Healthcare Settings"
(KNUST, 2026). Implementation and experiment notebooks to be added as the project
progresses through Phase 1 (baseline lock) and Phase 2 (S-FedProx engineering).
